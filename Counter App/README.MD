# Counter App 🚀

A fully responsive **HTML/CSS/JavaScript** counter application with color-coded display and mobile-first design.

## ✨ Features
- **Increment/Decrement** counter with single clicks
- **Color-coded display**: 
  - 🟢 **Green** for positive numbers
  - 🔴 **Red** for negative numbers  
  - 🔵 **Blue** for zero
- **Reset** functionality
- **Fully Responsive** - works perfectly on mobile & desktop
- **Modern UI** with Flexbox centering
- **No dependencies** - pure vanilla JavaScript


## 🛠️ Tech Stack
HTML5 | CSS3 | Vanilla JavaScript

## 🎮 How to Use
1. **Open** `index.html` in any browser
2. Click **Increment (+)** to increase count
3. Click **Decrement (-)** to decrease count  
4. Click **Reset** to set count to 0
5. Watch the color change automatically!
